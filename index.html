What Is SoccerAgent?

SoccerAgent is an agent that can be used to play football in robocup soccer simulations. It is based off the standard AgentBase in the TinMan framework but has a number of improvements aimed at making it easier to get started playing soccer!

Installation - Windows Only

Install Visual Studio 2012

Install the Visual Studio C++ 2008 Redistributable Package (x86)

Install simspark

Install rcssserver3d

Install Ruby

Restart your machine

Download and open the SoccerAgent Solution

Getting Started

To start programming your agent, you need to modify MySoccerAgent.cs

namespace Robocup
{
    class MySoccerAgent : SoccerAgentBase
    {
        public override void SoccerAgentThink(PerceptorState state)
        {
            switch (mState)
            {
                case State.IDLE:
                    break;
            }
        }
    }
}
