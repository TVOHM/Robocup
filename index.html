What Is SoccerAgent?

SoccerAgent is an agent that can be used to play football in robocup soccer simulations. It is based off the standard AgentBase in the TinMan framework but has a number of improvements aimed at making it easier to get started playing soccer!

Installation - Windows Only

Install Visual Studio 2012

Install the Visual Studio C++ 2008 Redistributable Package (x86)

Install simspark

Install rcssserver3d

Install Ruby (Check the option to add Ruby to PATH).

Restart your machine

Download and open the SoccerAgent Solution

Getting Started

To start programming your agent, you need to modify MySoccerAgent.cs

/// <summary>
/// Called every time the agent 'thinks'
/// </summary>
/// <param name="perceptorState">What the agent can percieve (i.e landmarks, ball position etc</param>
/// <param name="agentState">The current state of the agent (i.e. walking)</param>
public override void SoccerAgentThink
    (PerceptorState perceptorState, AgentState agentState)
{
    // Choose an action based upon the agent state
    switch (agentState)
    {
        case AgentState.IDLE:
            // The agent is doing nothing.
            // You may start a keyframe action - SetKeyFrame("kick");
            // Or tell the agent to walk - MoveTo(Landmark.FlagRightTop);
            //mWalk.MoveTo(perceptorState, Body);
            //SetKeyFrame("Fall_Forward"); 
            break;
        case AgentState.WALKING:
            // The agent is walking
            // You may update the walking behaviour - MoveTo(perceptorState.BallPosition);
            // Stop the walk - MoveStop();
            // Or interrupt the walk by starting a keyframe action - SetKeyFrame("kick");
            break;
        case AgentState.KEYFRAME:
            // The agent is performing a keyframe animation that was started by SetKeyFrame
            // You may not take any actions
            break;
        case AgentState.AUTORIGHTING:
            // The agent has fallen over and is in the process of getting up
            // You may not take any actions
            break;
    }
}
